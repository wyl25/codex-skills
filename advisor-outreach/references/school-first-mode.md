# School-First Mode

Use this mode when the user asks to find advisors by university rather than only by highest global research fit.

## Search Strategy

For each school, search across multiple units:

- Computer science, artificial intelligence, software, automation, electronic information.
- Biomedical engineering, medical school, imaging center, brain science, clinical AI institutes.
- Labs using terms such as medical image analysis, image segmentation, computer vision, pattern recognition, multimodal learning, neuroimaging, brain imaging, vessel segmentation, weak supervision, semi-supervision, AI healthcare, and intelligent diagnosis.

Do not limit the pool to medical imaging. Include adjacent computer vision and machine learning faculty when the user's CV can support the connection through Python, PyTorch, deep learning, image segmentation, model training, experiments, ablations, and visualization.

## School-Level Output

For each school, include:

- School type and relevant colleges/labs searched.
- A compact table of advisors grouped into S/A/B/C tiers.
- One-line reason for each advisor.
- Evidence link for each advisor when available.
- A short recommendation: first-contact list, backup list, and risky/low-fit list.

## Tiering

S tier:

- Directly matches weak/semi-supervised medical image segmentation, 3D vessel segmentation, brain image analysis, multimodal medical imaging, or medical foundation models.
- Best fit for deeply customized emails.

A tier:

- Medical image analysis, AI diagnosis, neuroimaging AI, multimodal medical data, or biomedical image computing.
- Strong outreach candidates even if the exact task differs.

B tier:

- Computer vision, segmentation, representation learning, multimodal learning, graph learning, robust learning, or trustworthy AI.
- Use the user's technical execution ability and CV/ML coursework as the email bridge.

C tier:

- Broad AI/data mining/pattern recognition/biomedical engineering with weaker direct evidence.
- Contact only when school preference is strong or the user wants a larger opportunity pool.

## Search Depth

For a first pass, target 5-12 relevant advisors per key school when available. If a school has fewer strong public matches, say so explicitly rather than filling with unrelated faculty.
