# Output Schema

Use these formats when producing advisor plans and batch emails.

## Advisor Ranking Table

| Priority | Tier | Advisor | School | School Type | College/Lab | Direction Fit | Evidence | User Angle | Email Focus | Risk/Gap |
|---|---|---|---|---|---|---|---|---|---|---|

Keep evidence concise and include links when available.

## Email Batch Format

For each advisor:

```text
【导师】姓名 - 学校 - 学院/课题组
【档位】A/B/C
【邮件主题】...
【个性化切入点】...

尊敬的X老师：
...
```

## Tracking Table

| Advisor | School | Tier | Email Sent | Reply | Follow-up Date | Notes |
|---|---|---|---|---|---|---|

## Missing Information Block

If inputs are incomplete, list only necessary missing items:

- Missing advisor homepage or paper links.
- Missing user's target degree/year.
- Missing preferred email length or tone.
- Missing attachment names.
