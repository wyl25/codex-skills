# Material Extraction

Extract a reusable profile from the user's CV, transcript, and template before writing emails.

## Profile Fields

- Name, current university, college, major, grade, target year.
- GPA, weighted average, required-course GPA, rank, class type, and course count above 90 if available.
- English scores: CET-4, CET-6, IELTS, TOEFL, GRE, or other evidence.
- Research groups, advisors, project topics, technical methods, datasets, models, and tools.
- Publications: author order, venue, status, title, exact contribution, and whether under review, accepted, or published.
- Patents: author order, title, status, and technical contribution.
- Competitions and awards: prioritize national/provincial awards and field-relevant awards.
- Core courses: select courses relevant to each advisor, not all courses.

## Advantage Bank

Create short reusable claims backed by evidence, for example:

- Academic stability: GPA/rank/core courses.
- Research fit: medical image analysis, weak supervision, 3D vessel segmentation, multimodal brain imaging, machine learning, or other user-specific directions.
- Execution ability: full experiment pipeline, comparisons, ablations, visualization, patent implementation.
- Mathematical/programming base: math competitions, Python/algorithm competitions, AI coursework.
- Communication and persistence: scholarships, student honors, long-term team participation.

Use only the advantages most relevant to each advisor. Do not list every achievement in every email.

## Transcript Use

Use transcript evidence to support fit when the advisor's direction needs specific foundations:

- AI/ML advisors: machine learning, deep learning, pattern recognition, data structures, algorithms, probability/statistics, optimization.
- Medical imaging advisors: computer vision, image processing, deep learning, biomedical-related courses if present.
- Brain imaging/multimodal advisors: statistics, signal processing, machine learning, data analysis, linear algebra.

Do not over-emphasize transcript details if research experience already provides stronger evidence.
